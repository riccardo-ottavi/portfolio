# Portfolio — Minimal Dark (JSX)

Versione standalone in **React + JSX puro** (no TypeScript, no backend), basata su Vite.

## Avvio
```bash
npm install
npm run dev
```
Apri http://localhost:5173

## Build di produzione
```bash
npm run build
npm run preview
```

## Struttura
- `index.html` — entry HTML con caricamento dei font Google
- `src/main.jsx` — bootstrap React
- `src/App.jsx` — pagina portfolio completa
- `src/styles.css` — design system minimale dark (CSS puro, senza Tailwind)
